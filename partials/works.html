Google Maps API – A Wrapper for .NET

Componente que fornece os métodos necessários para os serviços de Directions (rota) e Geocoding (localização de endereços). Utilização de WCF, RESTFul, JSON, .NET 4.0, Visual Studio 2010. No site você também pode obter o código-fonte. Mostro, inclusive, como obter um serviço de busca de CEP gratuito utilizando este componente.

Baixe a API

Google URL Shortener API for .NET

Componente que fornece os métodos necessários para encurtar URLs, além de obter as estatísticas fornecidas pela Google para os acessos através da URL encurtada. Utilização de WCF, RESTFul, JSON, .NET 4.0, Visual Studio 2010 e Mercurial. No site você também pode obter o código-fonte.

Baixe a API

Esta é uma ferramenta interessante tanto para futuras mamães e futuros papais, como para profissionais da área e estudiosos, pois além de trazer uma calculadora para calcular a idade gestacional baseada em diversos parâmetros, trás também uma série de informações adicionais para lhe ajudar em casa fase da gestação.

Metro São Paulo for Windows Phone

Finalmente chega ao Windows Phone uma versão exclusiva com informações sobre o transporte metropolitano de São Paulo. Este aplicativo foi caprichosamente desenhado para aproveitar o máximo de recursos do WP7, de modo que a navegação é uma experiência incomparável em relação a outros dispositivos móveis que não tenham o Windows Phone.

Tracking BR for Windows Phone

Você pode cadastrar seus objetos através do código de rastreamento de modo a facilitar futuras consultas, assim você não precisa ficar digitando o código todas as vezes. Além disto, caso as encomendas cadastradas tenham seu status alterado, ao entrar no aplicativo você será avisado – assim você não precisa ficar consultando os objetos o tempo inteiro para saber se houve alguma mudança.
